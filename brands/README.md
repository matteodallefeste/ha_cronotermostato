# Brand images

Placeholder (all-black) icons for the Weekly Thermostat integration.

Home Assistant and HACS do **not** load these files from this repository. To
make the logo appear on the integration card in Home Assistant, they must be
submitted to the [home-assistant/brands](https://github.com/home-assistant/brands)
repository under `custom_integrations/weekly_thermostat/`.

## Files

| File          | Size    | Notes                                  |
| ------------- | ------- | -------------------------------------- |
| `icon.png`    | 256x256 | Square icon (required)                 |
| `icon@2x.png` | 512x512 | hDPI icon (required alongside `icon`)  |
| `logo.png`    | 256x256 | Logo (optional)                        |
| `logo@2x.png` | 512x512 | hDPI logo (optional)                   |

Replace these black placeholders with the real artwork before submitting.

## Requirements (brands repo)

- PNG with transparency, trimmed of surrounding whitespace.
- `icon.png` must be square, between 128x128 and 256x256.
- `@2x` variants must be exactly double the base dimensions.
